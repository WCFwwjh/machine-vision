什麼是機器視覺檢測？

機器視覺檢測，又稱為 AOI（Automated Optical Inspection），是一種利用相機、光學設備與影像處理技術，自動擷取影像並分析影像內容的方法。

它可以用來判斷產品是否有瑕疵、尺寸是否正確、位置是否偏移，或是辨識條碼、文字與物件種類。

為什麼需要機器視覺？

⚡ 速度快

遠超人工目視的處理速度，適合高速產線。

🔒 穩定度高

不受疲勞、主觀判斷影響，結果一致可靠。

🔄 全天候運作

可長時間不間斷運作，適合 24 小時產線。

📊 客觀量化

判斷結果數據化，可追蹤品質趨勢。

02 AOI 定義與需求

AOI（Automated Optical Inspection）是現代工業製造的核心檢測技術。

AOI 定義

AOI 是以非接觸的方式自動擷取影像，並透過影像分析，從影像中抽取資訊，進一步控制程序或動作。

AOI 的需求

  • 自動化大量生產需要快速檢測，人工已無法跟上速度
  • 品質確認需要穩定且一致的判斷標準
  • 人工檢測容易受到速度、疲勞、主觀判斷影響，導致誤判

AOI 所需技術架構

🔧 硬體技術

光學設計、機台設計、運動控制、系統整合

💻 軟體技術

影像處理、特徵擷取、尺寸量測、瑕疵檢測演算法

AOI 的優勢

全自動化 檢測速度快 穩定度高 減少人工誤差 適合大量生產線 結果可量化

03 機器視覺功能

機器視覺在工業應用中具備六大核心功能，涵蓋從定位到量測的完整需求。

定位 Positioning

用來判斷工件的位置與方向。例如 PCB 板在機台上的位置、旋轉角度，讓系統可以自動對位。

識別 Recognition

根據物體本身的特徵判斷它是什麼。例如用形狀、顏色、紋理來識別不同零件。

辨識 Identification

透過符號、文字或編碼來判定物品身分。

Barcode QR Code OCR 車牌辨識

瑕疵檢測 Defect Detection

用來找出產品中不正常的部分。

PCB 斷線 / 短路 瓶子裂痕 異物 面板缺陷 木板坑洞

驗證 Verification

確認製程是否正確完成。例如貼標是否正確、零件是否裝上、包裝是否完整。

量測 Measurement

透過影像量測尺寸。

線寬 圓心與半徑 兩線距離 工件長寬

05 成像系統基礎

了解鏡頭成像原理、放大倍率與視野範圍的計算方式。

基本成像原理

鏡頭會把物體表面的資訊投影到感光元件上。成像後的影像會上下左右相反。

FOV 視野範圍

FORMULA
FOV = Sensor Size ÷ 鏡頭倍率
Sensor Size 增大
→ FOV 變大，可見範圍增加
鏡頭倍率增大
→ FOV 變小，解析度通常提高

Image Circle 與 Sensor Size

Image Circle 是鏡頭能投影出的影像範圍。感光元件必須小於或等於 Image Circle，否則可能出現邊角變暗或影像不完整（Vignetting）。

像素解析度

FORMULA
像素解析度 = Pixel Size ÷ 鏡頭倍率

06 相機規格

選擇相機時需要評估感光元件規格、快門、傳輸介面等多個面向。

相機選用評估要點

  • FOV：視野範圍是否足夠覆蓋檢測區域
  • 解析度：像素是否足夠偵測最小瑕疵
  • 速度：Frame Rate 是否符合產線速度
  • 傳輸介面：是否能支援所需資料量（GigE / USB / Camera Link）
  • 環境適應：是否適合工廠溫度、震動、粉塵環境

取像快門

Global Shutter

全域快門，整個感光元件同時曝光，適合高速或移動物體。

Rolling Shutter

滾動快門，逐行曝光，對快速移動物體可能產生扭曲。

常見傳輸介面

介面頻寬特點
GigE Vision~1 Gbps距離長，網路架構，易整合
USB 3.0~5 Gbps即插即用，適合中低速應用
Camera Link高速工業標準，高可靠性
CoaXPress超高速適合高解析度高速相機

07 鏡頭與成像系統選擇

鏡頭規格直接影響 FOV、解析度、景深與影像品質。

鏡頭接口

C Mount

法蘭距 17.526mm，工業相機最常見標準接口。

CS Mount

法蘭距 12.526mm，需注意與相機的相容性。

光圈 F#

F# 小（大光圈）
進光量多、景深淺，背景模糊，對焦精準度要求高
F# 大（小光圈）
進光量少、景深深，適合需要大景深的量測應用

Distortion 變形

鏡頭可能造成影像變形（桶狀或枕狀畸變）。量測應用中特別需要校正，否則會造成尺寸誤差。

焦距選用

焦距影響視野大小、工作距離與成像比例。短焦距適合大 FOV；長焦距適合遠距離或需要較小 FOV 的應用。

08 取像參數

正確設定取像參數是獲得高品質影像的關鍵，直接影響後續影像處理的效果。

光圈 Aperture

光圈控制景深與進光量，選擇需要在景深與亮度之間取得平衡。

曝光時間 Exposure

曝光時間控制進光量。時間越長影像越亮，但物體移動時曝光太長會造成模糊。

Gain / ISO

Gain 越大，畫面越亮，但雜訊也會同步增加，建議在不模糊前提下優先提高光源亮度，而非增大 Gain。

取像原則

  • 影像不要模糊（失焦或晃動造成的模糊幾乎無法補救）
  • 影像不要過曝（高光區域細節完全消失）
  • Gain 越小越好（減少雜訊干擾）
  • 曝光時間在不模糊、不過曝的情況下盡量足夠
  • 光圈依照景深需求調整

09 特殊成像系統

針對特殊檢測需求，工業視覺發展出遠心系統與線掃描系統。

遠心系統 Telecentric

遠心鏡頭的特色是等視角（Telecentric）。在尺寸量測時，物體高度變化不會影響橫向量測結果。

✅ 適合

高精度尺寸量測、工件高度可能有變化的情況

⚠️ 缺點

體積大、價格高，且物件不能超出鏡頭口徑

線掃描系統 Line Scan

當 FOV 很大，或產品在輸送帶上移動時，可以使用線掃描相機。它不是一次拍整張圖，而是一條線一條線掃描，再組合成完整影像。

大面積檢測 連續式產線 高速輸送帶 捲材檢測

10 照明系統

照明是 AOI 系統成敗的關鍵要素，好的照明設計能大幅提升瑕疵可見度。

照明的目的

  • 凸顯重要特徵，使瑕疵更明顯
  • 壓抑不重要的背景資訊
  • 增加缺陷與背景的灰階差異

光源種類

光源特點
鎢絲燈連續光譜，但高熱、耗電、壽命短
鹵素燈亮度高，常用於光纖導光，但同樣高熱
氙氣燈可做高速閃光，適合高速拍攝
螢光燈低熱，但頻閃可能影響成像
LED ⭐低熱、低耗電、壽命長、易控制，AOI 最常用

照明方式

明視野

光源從正面照射，呈現反射光差異。表面平整處亮、缺陷處暗。

暗視野

低角度側面照射，只有不平整處（如裂痕）才會散射光線而呈現亮區。

穿透光

光從背面穿透，呈現穿透率差異。適合透明/半透明物體檢測。

同軸光

光源與鏡頭同軸，適合鏡面或高反射表面，減少反光雜訊。

11 影像品質優化

了解影像品質指標與校正方法，確保影像資料可靠。

SNR 訊噪比

SNR（Signal-to-Noise Ratio）是衡量畫面純淨度的指標。SNR 越高，雜訊越少，影像品質越好。

Dynamic Range 動態範圍

衡量影像可呈現亮暗差異的能力。Dynamic Range 越大，可同時呈現明亮與暗部的細節。

白平衡

白平衡用來讓影像顏色正常，通常使用白色或灰色物體校正 RGB Gain，確保色彩還原正確。

亮度校正 FFC（Flat Field Correction）

用來修正畫面亮暗不均（Shading）的問題，確保整個視野的亮度一致。

對比擴張

透過調整 Offset 和 Gain，讓畫面黑白差異更明顯，提升後續影像分析的準確性。

彩色感光元件技術

Bayer Pattern 3CCD Bayer + Pixel Shift Foveon

12 影像處理基礎

影像處理是機器視覺的核心演算法基礎，涵蓋強度轉換、濾波、直方圖等技術。

灰階影像

灰階影像可以用 f(x, y) 表示，其中 x、y 是像素位置，f 是該位置的亮度值（通常為 0～255）。

強度轉換

方法用途
二值化將影像轉為黑白兩值，區分前景與背景
負轉換反轉亮度值，明暗互換
對數轉換增強暗部細節
指數轉換增強亮部細節

直方圖與等化

直方圖統計影像中每個灰階值出現的次數。直方圖等化（Histogram Equalization）可讓灰階分布更平均，提高影像對比度。

空間濾波器

濾波器功能
Average Filter平均鄰近像素，消除雜訊（但會模糊邊緣）
Gaussian Filter加權平均，較自然的模糊效果
Median Filter取中間值，有效去除椒鹽雜訊
Sobel Filter偵測邊緣方向（水平/垂直）
Laplacian Filter偵測邊緣，對方向不敏感

13 影像分割

影像分割是將影像切分成不同區域或物體，是瑕疵定位的重要前處理步驟。

分割方法

邊緣偵測

根據像素亮度的不連續變化來找出邊界。常用 Sobel、Roberts、Prewitt 運算子。

灰階相似

將灰階值相近的像素歸為同一區域。

顏色相似

根據顏色特徵分割，適合彩色影像。

紋理相似

根據紋理特徵分割，適合材質分析。

門檻化分割

利用亮度值把影像分成前景與背景。

類型說明
整體門檻化整張影像使用同一個門檻值
局部門檻化不同區域使用不同門檻值
動態門檻化門檻值依照座標或區域自動變化

Otsu 演算法

Otsu 是常見的自動門檻選擇方法，透過最大化類別間變異數（Between-class Variance），自動找出最佳門檻值，無需手動設定。

區域分割

  • 區域成長：從種子點開始，將性質相似的像素加入同一區域
  • 區域分裂：將影像切成小區域，判斷每個區域是否符合條件
  • 區域合併：將性質相似的區域合併成較大區域

14 形態學處理

形態學是用來處理影像形狀的工具，可簡化影像資料、保留重要形狀、去除雜訊。

基本運算

侵蝕 Erosion

縮小前景區域，常用來去除小雜訊或細小不重要區塊。

膨脹 Dilation

擴大前景區域，常用來連接斷裂區域或填補小縫隙。

斷開 Opening

先侵蝕再膨脹，去除小雜訊、斷開細連接，保留大區域形狀。

閉合 Closing

先膨脹再侵蝕，填補小洞、連接細小斷裂，保留整體形狀。

進階演算法

方法用途
Hit-or-Miss Transform形狀偵測，找出符合特定形狀的區域
輪廓萃取找出物體邊界
區塊填補填補前景區域內部的小洞
連通元件找出影像中相連的區塊，常用於瑕疵區域標記
凸形封包找出能包住物體所有點的凸多邊形
細線化把物體縮成較細的線條，保留原本形狀
骨架萃取找出物體的中心線，用骨架代表形狀

16 學習整理

重點複習與自我測驗，確認學習成效。

重點整理

  • AOI 是非接觸式自動影像檢測系統
  • 機器視覺適合大量、重複性、高穩定需求的工作
  • 成像系統包含相機、鏡頭、光源與取像參數
  • 照明會直接影響瑕疵是否能被看見
  • 影像處理包含濾波、增強、分割與形態學處理
  • 瑕疵檢測常需要結合多種方法（光源設計 + 演算法）

小考題

機器視覺的六大功能：定位（Positioning）、識別（Recognition）、辨識（Identification）、瑕疵檢測（Defect Detection）、驗證（Verification）、量測（Measurement）。
Q2 請簡述機器視覺檢測（AOI）的定義。
AOI 是以非接觸的方式自動擷取影像，並透過影像分析，從影像中抽取資訊，進一步控制程序或動作的系統。
Q3 請以魚骨圖方式總結機器視覺檢測所需的技術。
硬體：光學設計、機台設計、運動控制、系統整合。軟體：影像處理、特徵擷取、尺寸量測、瑕疵檢測演算法。中心節點：AOI 系統。
